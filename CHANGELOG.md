# Changelog

本项目的所有重要更新都会记录在这里。

格式参考 Keep a Changelog，并遵循语义化版本管理思路。

## [Unreleased]

## [v1.2.9] - 2026-07-10

### Changed

- **重新设计插件设置页**：常用配置前置，设置分组增加图标、清晰标题和用途说明，减少字段堆叠感
- **高级配置按需显示**：短视频、网易云、其他音乐平台和 HTTP 字段映射默认收起，通过“显示高级设置”开关展开
- **关联配置动态显示**：重试参数、语音限制、Alist 凭据、视频阈值、批量 JSON 阈值和调试长度仅在对应功能启用时显示
- **优化输入控件**：数值范围改用滑块与数字输入组合，请求方式增加可读下拉标签，JSON 参数启用代码编辑器
- **统一配置文案**：补齐字段名称、提示、建议范围和风险说明，移除未使用且会产生空白分组的缓存配置项

### Fixed

- **修复“附带原始分享链接”设置未生效**：开启后会在短视频文字信息末尾真实附加原始链接
- **修复关闭 Video 组件后的发送异常**：`pack_send_video=false` 时正确使用视频直链，不再因未初始化变量中断发送，也不会在元信息与资源节点中重复展示同一链接

## [v1.2.8] - 2026-07-10

### Fixed

- **修复停止解析偶发失效**：使用解析任务取消代次替代会被新任务重置的全局布尔标志，停止指令可稳定截断所有已启动的短视频与主页解析输出
- **修复合并转发与单条消息同时发送**：合并转发失败后不再逐作品降级发送，避免平台已接收打包消息但客户端抛错时产生混合输出
- **修复主页作品打包节点嵌套**：已构造的 `Node` 不再被重复包装，提升群聊合并转发兼容性

### Added

- **新增合并转发重试**：支持配置 `pack_retry_count` 与 `pack_retry_delay_ms`；重试全部失败后仅发送接口返回的完整 JSON
- **恢复插件打包脚本**：可使用 `python package_plugin.py` 生成 AstrBot WebUI 可直接上传的 zip

## [v1.2.7] - 2026-06-09

### Fixed

- **增强停止解析指令**：停止解析时自动清理缓存文件，并在发送过程中响应停止指令

## [v1.2.6] - 2026-06-08

### Fixed

- **增强解析锁机制**：为所有解析任务添加实例级解析锁，防止同一链接被多个事件处理器重复解析

## [v1.2.5] - 2026-06-08

### Fixed

- **修复重复发送问题**：解决命令处理器和自动触发同时执行导致重复发送消息的问题
- **修复变量引用错误**：修复视频链接变量引用不一致的问题

## [v1.2.4] - 2026-06-08

### Added

- **新增停止解析指令**：`bk停止解析` - 可强制结束当前解析任务并截断输出
- **新增批量内容输出控制**：图集/实况数量超过阈值时可直接输出JSON响应
  - `direct_json_enabled`：启用直接输出JSON功能
  - `image_count_threshold`：图集数量阈值（默认20）
  - `live_photo_count_threshold`：实况数量阈值（默认10）

## [v1.2.3] - 2026-05-16

### Added

- **视频文件大小阈值控制**：增加文件大小阈值判断机制，当视频文件超过设定值时自动发送资源链接而非直接传输文件
  - 新增 `video_threshold` 配置组，包含以下参数：
    - `force_link_enabled`：全局强制链接发送总开关（优先级最高，启用时始终发送链接）
    - `threshold_enabled`：阈值触发链接发送功能开关（仅在全局开关关闭时生效）
    - `threshold_mb`：文件大小阈值（MB），默认 100MB
  - 通过 HEAD 请求检测远程视频文件大小，避免不必要的下载开销
  - 完善的错误处理：文件大小检测失败时自动回退到正常发送流程

## [v1.2.2] - 2026-05-16

### Added

- **新增豆包（doubao.com）域名识别**：短视频自动解析支持识别豆包对话视频和图片链接
  - 支持 `/video-sharing` 路径的视频链接
  - 支持 `/thread/` 路径的对话图片链接
- **新增微信视频号（weixin.qq.com）域名识别**：短视频自动解析支持识别微信视频号链接
  - 支持 `/sph/` 路径的视频号链接

### Fixed

- **修复短视频打包功能间歇性失效问题**：简化 `_chain_to_forward_nodes` 函数，移除可能导致打包不一致的特殊分组逻辑

## [v1.2.1] - 2026-04-24

### Added

- **新增即梦（jimengai.com/klingai.com/jianying.com）域名识别**：短视频自动解析支持识别即梦视频链接

## [v1.2.0] - 2026-04-24

### Added

- **新增 Alist 上传功能**：视频先下载到服务器，再上传到 Alist，返回直链
- **新增 `upload_only` 配置**：可选择仅上传直链，或同时发送视频文件和直链
- **新增 Alist 配置分组**：地址、账号、密码、上传路径、分片大小等均可配置
- **新增 `_download_video` 函数**：支持大视频下载，带大小限制
- **新增链接去重功能**：短时间内相同链接只会解析一次

### Changed

- **修复 Alist API 调用**：改用 `PUT /api/fs/put` 接口上传文件
- 更新版本号至 v1.1.8
